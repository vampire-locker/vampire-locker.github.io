关于图片压缩工具

发表于2024-05-07|更新于2024-05-07

写在前面

鉴于我的个人博客的配置有限（当前是部署在 GitHub Pages 上），加上为了提高博客的加载速度，因此需要对博客的图片进行压缩。以下是我使用的压缩工具，在这里做下记录以便大家参考学习。

TinyPNG

TinyPNG 是一个在线的图片压缩工具，从它的主页可以看到一些介绍，这里我不再重复。
图片拖拽到网页上就可以直接进行压缩，单次支持最多20张图片，每张图片最大5MB。
下面是我的实测例子，由于是有损压缩，压缩效果比例还是挺可观的，对于观感也没有什么损失。

ImageOptim

ImageOptim 是一个开源的工具，可以下载到本地使用。
